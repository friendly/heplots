# Find degrees of freedom for model terms

Find degrees of freedom for model terms

## Usage

``` r
df.terms(model, term, ...)

# Default S3 method
df.terms(model, term, ...)
```

## Arguments

- model:

  A model object, such as fit using
  [`lm`](https://rdrr.io/r/stats/lm.html).

- term:

  One or more terms from the model

- ...:

  Other arguments, ignored
