- [`heplot()`](https://friendly.github.io/heplots/reference/heplot.md) :
  Two-Dimensional HE Plots
- [`heplot1d()`](https://friendly.github.io/heplots/reference/heplot1d.md)
  : One-Dimensional HE Plots
- [`heplot3d()`](https://friendly.github.io/heplots/reference/heplot3d.md)
  : Three-Dimensional HE Plots
- [`heplots-package`](https://friendly.github.io/heplots/reference/heplots-package.md)
  [`heplots`](https://friendly.github.io/heplots/reference/heplots-package.md)

  : Visualizing Hypothesis Tests in Multivariate Linear Models
- [`pairs(`*`<mlm>`*`)`](https://friendly.github.io/heplots/reference/pairs.mlm.md)
  : Pairwise HE Plots
- [`mark.H0()`](https://friendly.github.io/heplots/reference/mark.H0.md)
  : Mark a point null hypothesis in an HE plot

- [`plot(`*`<boxM>`*`)`](https://friendly.github.io/heplots/reference/plot.boxM.md)
  : Plot for Box's M test and generalizations
- [`eigstatCI()`](https://friendly.github.io/heplots/reference/eigstatCI.md)
